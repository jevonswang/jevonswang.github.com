安卓开发环境配置

  • android，Android Stdio
环境配置

##1. 安装jdk
（1）安装jdk
从java官网下载最新版的jdk安装包，并安装到制定目录下。在安装过程中，会先安装jdk，后安装jre，可以把它们分别装到不同目录下。
>
（2）设置环境变量

  • 右键我的电脑，点击属性 -> 高级系统设置 -> 环境变量。
  • 新建系统变量JAVA_HOME为E:\Program Files\Java\jdk1.8.0_66
  • 添加%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;到PATH变量中
  • 新建系统变量CLASSPATH为.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar
  • 运行cmd，输入javac -version，如果能够看到版本信息，说明安装成功。

##2. 安装android studio
（1）从安卓官网下载最新的android studio并安装。
（2）安装好以后打开，会自动下载Android SDK。

##3. 第一个项目
（1）打开Android Studio，选择Start a new Android Studio project。


（2）输入应用名称和项目存储位置。

  • Application Name 此处填写想呈现给用户的应用名称，此处我们使用“My First App”。
  • Company domain 包名限定符，Android Studio会将这个限定符应用于每个新建的Android项目。
  • Package Name 是应用的包命名空间（同Java的包的概念），该包名在同一Android系统上所有已安装的应用中具有唯一性，我们可以独立地编辑该包名。
  • Project location 操作系统存放项目的目录。

    （3）选择目标安卓设备。
    我们要为安卓手机开发程序，所以默认选择第一个。

    （4）选择添加第一个Activity

    （5）设置Activity的各种参数

    然后点击Finish，项目就建好了。
